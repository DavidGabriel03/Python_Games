	Aceasta este o implementare în Python a jocului piatră–foarfecă–hârtie, concepută pentru rularea în consolă. Programul permite utilizatorului să stabilească un număr-țintă de victorii consecutive necesare pentru câștigarea jocului. La fiecare rundă, jucătorul își introduce alegerea, iar adversarul (computerul) selectează aleatoriu una dintre opțiuni. Rezultatul este determinat prin compararea celor două alegeri: egalitate, victorie sau înfrângere. 
	În cazul unei victorii, contorul de runde crește; în caz de înfrângere, acesta se resetează. Jocul se încheie fie atunci când utilizatorul atinge numărul de victorii stabilit, fie când decide să nu mai continue.
