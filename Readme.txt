Hangman este un joc clasic implementat în Python pentru consolă, unde utilizatorul trebuie să ghicească un cuvânt ales aleatoriu dintr-o listă. La fiecare încercare greșită, desenul ASCII al spânzurătorii se completează progresiv, iar jocul se încheie fie când cuvântul este descoperit, fie când se epuizează numărul de încercări disponibile.
